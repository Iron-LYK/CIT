# CIT
This repo is the offical PyTorch implementation of “**Cascaded Iterative Transformer for Jointly Predicting Facial Landmark, Occlusion Probability and Head Pose**”

In this paper, we propose a **transformer-based sota algorithm** for Facial Landmark Detection, Occlusion Probability Estimation and Head Pose Estimation. **A new dataset** is likewise proposed.
